Random Job Generator Web Application 
Developed a Random Job Generator web application using HTML, JavaScript, and External CSS that dynamically displays job roles on user interaction.
The project focuses on DOM manipulation, event handling, and clean UI styling to enhance user experience.
Tools & Technologies Used:
1.HTML5 – Structure of the application
2.JavaScript (ES6) – Logic, random selection, DOM manipulation
3.External CSS3 – Styling, layout, responsiveness
4.VS Code – Code editor
5.Git & GitHub – Version control and project hosting
